# Pharmacy_Management_System_OOP2

Project Overview
The Pharmacy Management System is a C#-based application developed to manage pharmacy operations efficiently. The system focuses on medicine management, user handling, and billing processes using object-oriented programming principles.

This project is designed as an academic project for OOP2, demonstrating clean code structure, role-based access control, and real-world pharmacy workflows.

User Roles
The system supports three types of users with role-based login:

- Admin
  - Manage employees and customers
  - Manage medicines (add, update, delete)
  - View and manage bills

- Employee
  - Manage medicines
  - Create bills for customers
  - Assist daily pharmacy operations

- Customer
  - Login to the system
  - View available medicines
  - Purchase medicines and receive bills

Core Features
- Role-based authentication system  
- Medicine management 
- Billing and invoice generation  
- User management (Admin, Employee, Customer)  
- Secure and structured data handling  
- Object-Oriented Programming2 (OOP2) based design  

Technologies Used
- Programming Language: C#  
- Framework: .NET  
- Database: SQL Server  
- Architecture: Object-Oriented Programming2 (OOP2)  
- Tools: Visual Studio
